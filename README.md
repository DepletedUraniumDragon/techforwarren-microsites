# Microsites

Spin up a Next.js + React site quickly!

## CLI

### Help

    bin/cli --help

### Clone a site

    bin/cli clone [site]

`site` is the name of the new site. It will be available in `src/sites/[site]`.

### Start a site

Run a site locally.

    bin/cli start [site]

The name of the site to run.

Alternatively you can:

    cd src/sites/[site] && npm run start:dev
